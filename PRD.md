# 易经起卦小工具产品需求文档
## 一、产品定位
纯游客模式、现代极简明亮风格的易经起卦工具，面向年轻用户，操作简洁、结果直白，无登录、无广告、无复杂付费逻辑，仅提供基础起卦与分享功能。

## 二、核心功能与使用规则
### 1. 首页布局
- 首页展示**两个并列主入口**：
  1. 铜钱起卦（普通起卦）
  2. 每日一卦

### 2. 普通铜钱起卦
- 每日次数限制：**每日严格 3 次**，用完按钮置灰不可点击，无广告/付费补充次数
- 次数周期：自然日 0 点自动重置
- 问题类型：默认「通用」，支持手动切换：通用 / 事业 / 感情 / 财运 / 学业
- 切换问题类型：**重新随机生成一卦**
- 起卦方式二选一：
  - 系统模拟：一键直接生成完整卦象
  - 手动起卦：一次性展示 6 行爻位选择，逐行完成选择
- 结果展示：
  - 年轻化简约图形爻象（非传统阴阳横线）
  - 动爻使用**小色块/小图标**视觉标记
  - 明确标注：吉 / 平 / 凶
  - 固定解读文案：**40～60 字**
- 分享功能：生成**渐变色年轻风格卡片**，仅展示「卦名 + 白话解读」

### 3. 每日一卦
- 每日次数限制：**仅 1 次/天**
- 次数与普通铜钱起卦**独立计算，互不占用**
- 支持用户自选问题类型：通用 / 事业 / 感情 / 财运 / 学业
- 起卦逻辑、结果结构与普通起卦一致
- 当日已占：直接展示结果，不可重复起卦

### 4. 全局通用规则
- 纯游客模式，无需登录、注册
- 次数与记录存储在**设备本地**，不上传服务器
- 无广告、无内购、无额外次数解锁
- 支持“一事不二占”文案提示，不做强制限制

## 三、页面结构与交互流程
### 1. 首页
- 顶部：应用名称 + 简洁标语
- 中部：两个等大按钮「铜钱起卦」「每日一卦」
- 底部：轻量说明（每日次数规则、无广告提示）

### 2. 铜钱起卦页面
- 顶部：今日剩余次数 `X/3`
- 中部：问题类型切换栏（通用/事业/感情/财运/学业）
- 操作区：
  - 系统模拟起卦按钮
  - 手动起卦按钮
- 手动起卦面板：6 行爻位选择区，一次性展示并完成选择
- 结果页：
  - 简约图形卦象 + 动爻标记
  - 卦名、吉凶标签
  - 40～60 字解读文案
  - 分享按钮

### 3. 每日一卦页面
- 未起卦状态：选择问题类型 → 起卦 → 展示结果
- 已起卦状态：直接展示当日卦象结果，不可再次操作
- 结果结构与普通起卦完全一致

### 4. 分享卡片
- 风格：渐变色年轻风
- 内容仅包含：卦名 + 白话解读

## 四、UI/UX 设计规范
- 整体风格：**现代极简明亮**
- 爻象展示：使用简约方块/圆点等年轻化图形，不使用传统阴阳爻横线
- 动爻标记：小色块 / 小图标形式
- 文案风格：直白易懂、口语化、不晦涩
- 按钮状态：次数用完后直接置灰，不可点击并提示次数已用尽

## 五、本地数据存储方案（游客模式）
使用本地存储记录以下核心字段：
- `daily_normal_count`：当日普通起卦次数
- `daily_normal_date`：普通起卦日期标记
- `daily_one_used`：当日每日一卦是否已使用
- `daily_one_date`：每日一卦日期标记
- `daily_one_result`：当日每日一卦结果缓存

逻辑规则：
- 每次启动对比当前日期与记录日期，日期不同则重置对应次数
- 纯本地运行，无网络也可正常使用
- 清除缓存会导致次数记录重置

## 六、文案格式规范
### 结果页标准格式
```
【卦名】XXXX
【吉凶】吉 / 平 / 凶
【解读】（40–60 字固定文案，说明趋势+简要建议）
```

### 分享卡片仅展示
```
【卦名】XXXX
解读内容……
```

## 七、边界与异常处理
- 清除应用缓存 = 本地记录重置，次数恢复初始状态
- 跨日自动刷新次数，无需手动操作
- 无网络环境可完整使用
- 暂不提供历史卦象记录功能
- 切换设备视为新用户，次数独立计算
